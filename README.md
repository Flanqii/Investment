# Risk Parameters Definition

This document defines the risk evaluation framework.

## Metrics Used

| Metric | Description |
|------|------------|
| CAGR | Annualised growth rate |
| Volatility | Annualised volatility |
| Max Drawdown | Worst peak-to-trough loss |

## Risk Interpretation

- High CAGR + Low Volatility → Stable growth
- High Volatility → Risky asset
- Large Drawdown → Poor downside protection
